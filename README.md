# gi: a command for 2fa
*G*et *I*nteger

## commands
run the following commands with `-h` option to learn how to use.
`gi run` `gi list` `gi add` `gi delete`
## build
```
git clone https://github.com/sharpedmimishee/gi
cargo build
```
